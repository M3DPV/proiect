Ce este printarea 3d?

Printarea 3d este un proces prin care se realizeza corpuri tridimesionale.Acest proces este unul aditiv,prin care se adauga strat cu strat material.Printarea 3d poate inlocui procesele traditionale care implica:taiat,forajare etc.

Ce materiale se pot folosi?

ABS

ABS este un material dur,care poate rezista la temperaturi extreme.

ABS nu este biodegradabil si se mincsoreza la conatctul cu aerul.

ABS este produs printr-un oroces numit emulsie,prin acelasi proces este produs si laptele.

Temperatura de printare este de 230ºC-260ºC.

PLA

PLA cunoscut ca si acid polilactic este un material biodegradabil.

PLA poate fi utilizat si in producerea de haine,tavi pentru microunde si in productia plasticului.

PLA este un material mai dificil de manipulat din cauza vitezei mari de racire si solidificare.

Temperatura de printare este de 190ºC-230ºC.

ULTEM

ULTEMul este un material din plastic caracterizat prin greutate redusa si elastcitate.

Rezista la temperaturi intre -30°-170°.

Acest material nu se decoloreaza la actiunea razelor UV.

Acesta este folosit la dispozitive medicale high-tech datorita proprietetilor termoplastice si electrice.

Care sunt evantajele printari 3d?

Productie mai rapida fata de metodele de productie traditionale.

Este usor de accesat,chiar si pentru oameni obisnuit,software-urile fiind open-sourse.

Este mai putin costisitor,deoarece deseurile de materii prime fiind reduse.

Istoria printari 3d.

Introducere

  • Printare 3d nu este un proces tocmai nou,aceasta tehnologia s-a popularizat in jurul anului 2009,de catre FDM.
  • Dar acest proces este mai timpuriu,prima imprimanta 3d fiind construita in 1980.

Anii 1980

  • Imprimantele 3d au fost imaginate inca din 70,dar au fost construite in anul 1981 de catre doctorantul japonez Hideo Kodoma.
  • Primul sistem de prototipare comerciala, a fost introdus in 1987.

Anii 1990

  • La inceput anilor 90 tot mai multe companii au inceput sa se bata pe piata prototipari 3d.
  • Stratasys a adus printarea 3d in casele oamenilor de rand.
  • In anul 1999 a fost prototipat primul organ.

Anii 2000-prezent

  • Primul rinichi a fost printat.
  • Organovo,descopera noi tehnici in domeniul bioprintingului.
  • Primele automobile sunt printate.
